## 🚀 Featured Projects

<details>
<summary><strong>📦 Dhaka Local Fare — Transport Fare Discovery App</strong></summary>

<br/>

A production-grade, cross-platform transport fare discovery application built for the 48 neighborhoods of Dhaka. Designed to give commuters instant, real-time access to local fare information across web and mobile.

| Attribute | Details |
|-----------|---------|
| **Stack** | React Native (Mobile), React (Web), Firebase |
| **Scale** | 48 Dhaka Neighborhoods |
| **Performance** | Real-time pricing with Firebase Realtime Database |
| **Security** | Firebase Authentication & Security Rules |
| **Impact** | Addresses transport transparency for millions of Dhaka commuters |
| **Repository** | [![GitHub](https://img.shields.io/badge/View%20Repo-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/ragibulfat) |

This project demonstrates my ability to architect cross-platform solutions from scratch — managing a shared data layer across both a React Native mobile client and a React web frontend. Firebase serves as the backbone for authentication, real-time data sync, and hosting. The system is scoped to serve real civic utility at the city level.

</details>

<details>
<summary><strong>📚 Book Shop Management System — Java Desktop Application</strong></summary>

<br/>

An enterprise-style desktop application for managing bookshop operations end-to-end — built with Java and integrated with a relational MySQL backend.

| Attribute | Details |
|-----------|---------|
| **Stack** | Java (Swing/OOP), MySQL |
| **Scale** | Single-store inventory management |
| **Performance** | Optimized SQL queries with normalized schema |
| **Security** | Role-based access, input validation |
| **Impact** | Automates inventory, billing, and customer record management |
| **Repository** | [![GitHub](https://img.shields.io/badge/View%20Repo-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/ragibulfat) |

The MySQL integration uses normalized relational schemas with efficient query design, making the system both robust and extensible.
